🌤️ Weather Dashboard

A weather application that fetches real-time weather data using the Open-Meteo API (no API key required). Users can search by city name or use their current location.

📌 Live Demo
View Live Demo at "https://checkingweather.netlify.app/"

🎯 Features

1. Search by City: Enter any city name to get current weather
2. Current Location: Uses browser geolocation to show local weather
3. Current Conditions: Temperature, humidity, wind speed, feels-like temperature
4. Weather Icons: Dynamic emoji icons based on weather conditions
5. Error Handling: User-friendly messages for invalid cities or network errors

🛠️ Technologies Used

| Technology | Purpose 
|------------|---------
| HTML5      | Page structure 
| CSS3       | Styling, responsive design 
| JavaScript | API calls, geolocation, DOM manipulation 
| Open-Meteo API | Free weather data (no API key) 
| OpenStreetMap API | Reverse geocoding (city names from coordinates)
